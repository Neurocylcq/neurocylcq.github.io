# Writing Standards

This page defines the difference between Notes and Blog content.

<!-- truncate -->

## Notes

- Goal: speed and utility
- Verification level: lightweight
- Status: may include incomplete or incorrect conclusions

## Blog

- Goal: publish reliable technical writing
- Verification level: stronger checking and revision
- Expected quality: references, traceable reasoning, clearer claims

## Practical Rule

If an idea is still evolving, keep it in Notes.

If a topic is stable and can be defended with evidence, publish it as a Blog post.
