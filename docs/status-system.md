# Note Status System

Use status labels to expose confidence and maturity.

## Status Definitions

<span class="note-status note-status--draft">Draft</span>

- In-progress ideas
- May contain incorrect conclusions
- Prioritize speed over completeness

<span class="note-status note-status--reviewed">Reviewed</span>

- Rechecked with references or experiments
- Claims are clearer and more stable
- Suitable for repeated reuse

<span class="note-status note-status--archived">Archived</span>

- Kept for historical context
- May be outdated for current practice
- Not recommended as primary guidance

## Practical Rule

Start everything as Draft. Promote to Reviewed when confidence is sufficient. Mark Archived when superseded.
