---
title: 词嵌入-Token Embedding
description: 词嵌入
---

词嵌入（Token Embedding）是将分词后的离散 Token 转换为连续低维向量的过程，每个向量都会携带对应 Token 的语义信息，是大模型能够理解文本的基础。

这一过程本质上是查表操作：模型内部维护了一张词向量表，每个 Token 对应表中一行固定维度的向量。

这张表本身就是模型的可学习参数，会在训练过程中不断更新优化。

它并不是一个独立复杂的神经网络，而是模型的嵌入层（Embedding Layer），输入为 Token ID，输出为对应的高维向量表示。

这些参数会伴随整个 LLM 一起，在海量文本数据的训练中逐步学习并收敛，最终让语义相近的 Token 拥有更接近的向量。
